How to effect Hue, Saturation, Lightness and Alpha

  • Hue - Is expressed as an angle, 0 through 360
  • Saturation - Is expressed as a percentage
  • Lightness - Is expressed white to black, 0% to 100% respectively
  • Alpha - Is expressed as a number for transparency, 0 through 1.0 respectively
